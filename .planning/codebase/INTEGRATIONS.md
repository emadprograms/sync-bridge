# Integrations

**Date:** 2026-07-12

## Current Integrations
- **SMB Networking**: The project interacts heavily with native Windows SMB network paths (e.g., `\\BridgeSync`).
- **File System**: Direct Windows IO for logging and file movement.

There are currently no external API integrations, databases, or webhooks configured.
